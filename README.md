# 我的电视

安卓电视直播软件，内置直播源

## 使用

下载安装 [releases](https://github.com/lizongying/my-tv/releases/)

其他地址 [my-tv](https://lyrics.run/my-tv.html)

## 其他

小米电视可以使用小米电视助手进行安装

如电视可以启用ADB，也可以通过ADB进行安装：

```shell
adb install my-tv.apk
```
